\doxysubsection{Detailed Description}
{\ttfamily \mbox{\hyperlink{class_arguments_line_edit}{Arguments\+Line\+Edit}}} is a customized {\ttfamily QLine\+Edit}. 

It offers over the Qt\textquotesingle{}s widget, is that we can handle some special characters and emit signals that are usefull elsewhere.

Since this widget can both display and edit the driver\textquotesingle{}s arguments, one could by mistake edit the symbol\textquotesingle{}s name here, confusing the program that reads this name from another input widget. Clicking or typing at a certain position in the arguments list, is prohibeted and the user is visually informed about the error.
