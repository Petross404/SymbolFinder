Detailed Description

Scanner class creates an object that can hold an IDriver* instance.

Scanner inherits the QObject class and defines the interface that gives the MainWindow the ability to input and output information to/from the supported drivers.

As of now, the only supported driver is nm.

Todo:
Add support for scanelf as well.
